each array is as follows :<br/>
'ispc' =ISPC<br/>
'dec' = DEC<br/>
'usp' = Unique name of the signalling point<br/>
'nsp' = Name of the signalling point operator<br/>
<br/>
some usp's are empty so replaced with # tag<br/>
